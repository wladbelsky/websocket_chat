# Тестовое задание

## Установка и запуск

1. Склонируйте репозиторий:
2. Запустите проект:
```bash
docker-compose up
```
3. (Опционально) Заполните базу данных тестовыми данными автоматически:
```bash
./create_test_users.sh
```

Так же если используете не стандартный порт, то для скрипта `create_test_users.sh` нужно будет указать его в переменной `PORT`:
```bash
PORT=8080 ./create_test_users.sh
```

4. Используйте любой WS клиент для подключения к серверу, данные для подключения будут отображены в терминале после запуска скрипта (Или могут быть получены вручную по API)
Формат сообщения:
```json
{
	"type": "message",
	"content": "test"
}
```
Формат сообщения об прочтении:
```json
{
	"type": "read_status",
	"message_id": 2
}
```
